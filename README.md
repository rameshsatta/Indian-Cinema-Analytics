Viewers having access to multiple streaming platforms, each offering vast movie libraries.

However, they struggle to decide which platform features the best-rated, most recent, or most engaging movies.

This abundance of choice makes it difficult for users to discover quality content efficiently.

The project aims to analyze Indian movie data to uncover patterns in release trends, runtime, genres, and ratings.

The goal is to provide data-backed insights that help understand audience preferences and content engagement patterns.
